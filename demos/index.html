Webshims Lib

About

Webshims Lib enables some of the latest Browserfeatures also known as "HTML5-Features" in non capable Browsers. It is build on Top of jQuery.

Main concepts

  • Capability based loading: lightweight for good browsers / heavyweight for really bad browsers (I really like this)
  • almost pure native HTML5 | no/less abstractions: Helps you to learn HTML5 (Markup-/DOM-APIs) and to build usefull abstractions and extensions on top of the standard. Documentation is the HTML5 W3C Specification
  • futureproof Feature-Detection: We try to feature detect everything instead of browsersniffing to be more futureproof
