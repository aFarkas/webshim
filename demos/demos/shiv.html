Demo/About/Documentation HTML5 shiv

IE8- isn't capable of rendering HTML5 elements. Webshims lib in combination with modernizr automatically fixes this issue.

Additionally it implements the following manipulation methods .htmlWebshim(), .appendWebshim(), .prependWebshim(), .afterWebshim(), .beforeWebshim() and .replaceWithWebshim() to also allow dynamic creation of HTML5 elements.

$('#my-id').appendWebshim('<section><form action="#"><input type="text" placeholder="hello this is required" required="required" name="test" /><input type="submit" /></form></section>');

Note: There are reasonable concerns against using this or other technics to implement styleable HTML5 elements. Before you use HTML5 semantics, you should know the problems and make a cost-benefit analysis for your project.
